# COD2 Mods Wiping

*Description*:
> That script remove your modded scripts from your Call Of Duty 2 installation PATH.
>
> Note that only *.iwd files are processed.
>
> Folders, *.cfg, *.dat, *.log, *.txt, ... are ignored.
>
